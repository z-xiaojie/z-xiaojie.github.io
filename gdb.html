Linux g++ Debug Guide
  • g++ -g source -o dest
  • type gdb to enter debug model
  • press l , list your code
  • press b linenumber, to make break point
  • press b function name, to make break point
  • press r, to run your code
  • press n, to debug step by step
  • press s, to enter a function
  • type "print a", to show the value of a.
  • press q, to quit debug model
